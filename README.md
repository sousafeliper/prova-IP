# 🧠 Soluções: Prova de Introdução à Programação (BIA - UFG)

Este repositório reúne as resoluções desenvolvidas para a avaliação prática da disciplina de **Introdução à Programação**, integrante do 1º período do bacharelado em **Inteligência Artificial** na Universidade Federal de Goiás (UFG).

Sob a orientação do **Professor Leonardo Alves**, os desafios abordam desde fundamentos de lógica até a integração com APIs externas, análise de dados e interfaces gráficas.

---

## 🎓 Contexto Acadêmico

O projeto reflete o primeiro contato prático com o ecossistema Python no curso, demonstrando a aplicação de algoritmos em problemas do mundo real e a base necessária para o desenvolvimento de soluções voltadas à IA.

---

## 📂 Conteúdo do Repositório

A prova é composta por seis módulos que escalam em complexidade:

* **Questão 1: Cálculo de Consumo Energético**
* Script para validação e cálculo de gasto mensal (kWh) com base na potência e tempo de uso.


* **Questão 2: Gestor Financeiro com Metas**
* Sistema de controle de despesas categorizadas, com comparação em tempo real entre gastos efetuados e metas estabelecidas.


* **Questão 3: Análise Estatística de Temperaturas**
* Uso de **NumPy** e **Pandas** para processar dados climáticos, gerando médias, desvios padrões e classificações térmicas.


* **Questão 4: Engine de Lógica (Tetris-Style)**
* Implementação de lógica de blocos e manipulação de matrizes utilizando DataFrames para simular a mecânica de um jogo de encaixe.


* **Questão 5: Concepção do SkillScan**
* Documentação e arquitetura de um analisador de currículos automatizado que utiliza as APIs do **GitHub** e do **Gemini** para avaliar perfis técnicos.


* **Questão 6: Dashboard de Cotação em Tempo Real**
* Interface gráfica (GUI) desenvolvida com **Tkinter** que consome dados de uma API externa de câmbio para exibir a cotação USD/BRL e histórico recente.



---

## 🛠️ Tecnologias e Bibliotecas

Para a resolução dos problemas, foram exploradas ferramentas essenciais da stack de dados e desenvolvimento Python:

* **Lógica e Estruturas:** Python 3.10+
* **Manipulação de Dados:** NumPy e Pandas
* **Interface Gráfica:** Tkinter
* **Comunicação Web:** Requests (Consumo de APIs REST)
* **Back-end (Proposta):** Flask e PyGithub

## 🚀 Como Executar

```bash
git clone https://github.com/sousafeliper/prova-IP.git

```


2. **Instale as dependências:**
```bash
pip install numpy pandas requests

```


3. **Execute um dos módulos (exemplo: Questão 6):**
```bash
python questao6.py

```
