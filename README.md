🧠 Prova de Introdução à Programação
Um repositório que reúne soluções de problemas clássicos de programação, demonstrando lógica afiada e domínio de Python.

📚 Sobre o Projeto
Este repositório contém soluções para questões de uma prova de Introdução à Programação. Cada script aborda um problema específico, com foco em lógica de programação e boas práticas de codificação.
